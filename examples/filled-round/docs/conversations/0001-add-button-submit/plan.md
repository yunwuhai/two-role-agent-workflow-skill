# 当前计划

## 目标

为待办输入框增加按钮提交流程。

## 背景

项目当前只能通过外部调用创建待办，尚无直接的输入组件提交入口。

## 允许修改文件

- `src/TodoInput.tsx`
- `src/TodoInput.test.tsx`

## 各文件任务

- `src/TodoInput.tsx`：增加输入框和按钮提交流程。
- `src/TodoInput.test.tsx`：增加按钮提交测试。

## 实施步骤

1. 增加输入框和提交按钮。
2. 复用已有 `onSubmit` 回调。
3. 增加测试覆盖。

## 接口 / 数据约束

- `onSubmit` 保持接收字符串。

## 验收标准

- 输入非空内容后点击按钮，会触发提交。
- 空字符串不会被提交。

## 非目标

- 不增加键盘快捷键。
