# 当前计划

## 目标

为 `TodoInput` 增加按钮提交能力，并补齐最小测试覆盖。

## 背景

当前组件只支持 Enter 提交，用户希望移动端也能通过按钮完成同一动作。

## 允许修改文件

- `src/components/TodoInput.tsx`
- `src/components/TodoInput.test.tsx`

## 各文件任务

- `TodoInput.tsx`：复用现有提交逻辑并新增按钮入口。
- `TodoInput.test.tsx`：覆盖按钮点击成功提交和空输入不提交。

## 实施步骤

1. 提取或复用现有提交处理逻辑。
2. 增加一个可访问的提交按钮。
3. 为按钮路径增加测试。

## 接口 / 数据约束

- 不改变现有 `onSubmit` 的入参形状。
- 不新增外部依赖。

## 验收标准

- Enter 与按钮点击都能提交非空文本。
- 空输入不会触发 `onSubmit`。
- 相关测试通过。

## 非目标

- 不重构整个表单布局。
- 不增加新的样式系统。
