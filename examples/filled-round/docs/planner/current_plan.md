# 当前计划

## 目标

为待办输入框增加“按 Enter 提交”能力。

## 背景

上一轮已完成新增按钮提交流程。本轮只补充键盘交互，不改变列表渲染逻辑。

## 允许修改文件

- `src/TodoInput.tsx`
- `src/TodoInput.test.tsx`

## 各文件任务

- `src/TodoInput.tsx`
  - 在输入框中增加 Enter 键提交行为
- `src/TodoInput.test.tsx`
  - 增加 Enter 提交的测试

## 实施步骤

1. 为输入框增加键盘事件处理。
2. 复用现有提交逻辑，避免复制业务分支。
3. 增加测试并确认现有测试不回归。

## 接口 / 数据约束

- 不修改 `onSubmit` 的函数签名。
- 空字符串仍不得被提交。

## 验收标准

- 输入非空文本后按 Enter，会触发一次提交。
- 输入为空时按 Enter，不会触发提交。
- 现有按钮提交流程测试仍通过。

## 非目标

- 不增加快捷键配置。
- 不调整待办项样式。
